* Panel con acciones prefijadas
Cuando seleccionems solamente una de las estructuras, hemos de mostrar el panel con acciones prefijadas. Es decir, cuando mostremos /All/ mostramos el campo para entrada con texto, cuando mostremos una estructura concreta solo el panel de ayuda.
Las acciones prefijadas no se pueden hacer con texto, sino usando la API de alto nivel de /redis-rust/.

*** Strings
Comandos a poner: /get/, /set/, /getset/[fn:1], /mget/, /mset/, /incr/, /decr/, /incrby/, /setnx/[fn:2], /getrange/, /setrange/, /ttl/ , /pttl/, /exists/, /append/, /getdel/, /strlen/
